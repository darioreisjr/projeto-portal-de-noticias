# Portal de Notícias de Tecnologia 🚀

Bem-vindo ao repositório do **Portal de Notícias de Tecnologia**, um projeto desenvolvido durante a formação Full-Stack da Rocketseat. Esta homepage foi criada para oferecer uma experiência intuitiva e moderna para usuários de desktop, destacando as últimas novidades em tecnologia, inteligência artificial, robótica e inovações digitais.

## ✨ Recursos Principais
- **Interface responsiva para desktop** com layout organizado e moderno.
- **Seções dinâmicas**:
  - Notícias sobre **Inteligência Artificial**, **Robôs Domésticos**, **Previsão de Eventos Futuros** e **Tecnologia para Idosos**.
  - Destaques de investimentos, projetos inovadores e tendências tecnológicas.
  - Cards informativos com imagens e textos otimizados.
- **Design System** com cores, tipografia e componentes padronizados.
- **Navegação intuitiva** com ícones e botões interativos.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** para estruturação semântica.
- **CSS3** com arquivos modularizados:
  - Estilos globais (`global.css`).
  - Estilos específicos para header, seções e utilitários.
- **Git** para controle de versão.
- **Vercel** para deploy automatizado.

## 🎨 Design System
### Cores
| Tipo            | Código    |
|-----------------|-----------|
| Primária        | `#4049FA` |
| Secundária      | `#104E0B` |
| Fundo           | `#0977A`  |
| Texto Primário  | `#FF5F`   |
| Texto Secundário| `#CBD5EI` |

### Tipografia
- **Fonte Principal**: `Archivo`
  - Títulos: `24px` a `14px` com pesos variados (Semibold, Regular).
  - Parágrafos: `14px` com espaçamento de linha de `160%`.

### Ícones
- Ícones SVG para navegação (`ArrowRight`, `MagnifyingGlass`, `List`).

## 📂 Estrutura do Projeto

Este projeto segue uma estrutura organizada para facilitar a manutenção e escalabilidade do código.

## Diretórios e Arquivos

### 📂 projeto-portal-de-noticias/
- **📂 Assets/** → Contém recursos visuais, como ícones e imagens.
  - **📂 Icons/** → Ícones SVG utilizados na interface.
    - 📄 ArrowRight-hover.svg → Ícone de seta ao passar o mouse.
    - 📄 ArrowRight.svg → Ícone de seta padrão.
    - 📄 ... (outros ícones)
  - **📂 Images/** → Imagens utilizadas no portal.
    - 📄 Image 01.png → Exemplo de imagem.
    - 📄 Image 02.png → Outra imagem de exemplo.
    - 📄 ... (outras imagens)

- **📂 styles/** → Arquivos de estilos CSS do projeto.
  - 📄 global.css → Estilos globais aplicados ao site.
  - 📄 header.css → Estilos específicos para o cabeçalho.
  - 📄 sections.css → Estilos para as seções do portal.
  - 📄 ... (outros estilos)

- **📄 index.html** → Arquivo principal da aplicação contendo a estrutura HTML.
- **📄 .gitignore** → Define arquivos e diretórios que devem ser ignorados pelo Git.
- **📄 README.md** → Documentação do projeto, explicando sua estrutura e uso.
